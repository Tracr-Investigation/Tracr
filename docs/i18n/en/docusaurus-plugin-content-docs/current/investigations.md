# Investigations

An investigation is the main container in Tracr. It groups together documents, tasks, entities, and a map.

## Create an investigation

1. From the **Dashboard** or the **Investigations** page, click **New investigation**.
2. Enter the name, description (optional), and select a status.
3. Confirm — you are automatically added as the owner.

## Statuses

Statuses are configurable by an administrator (Administration panel → Statuses). Each investigation carries a status visible from the list. Statuses allow filtering and sorting investigations.

## Collaborators

To add a collaborator to an investigation:
1. Open the investigation → any tab.
2. Click the **Collaborators** icon in the header.
3. Search for a user by their username and add them.

Collaborators have access to all tabs of the investigation. Only the owner can delete it.

## Investigation tabs

| Tab | Content |
|---|---|
| **Documents** | List and editing of the investigation's documents |
| **Graph** | Visualization of entities and their relationships |
| **Map** | Interactive map of geolocated entities |
| **Tasks** | List of the investigation's tasks |
| **Timeline** | Timeline of logged events |

## Delete an investigation

Deletion is permanent and removes all associated documents, tasks, and entities. Only the owner can perform this action.
