# Welcome to Tracr

Tracr is a collaborative investigation management platform. It enables teams to conduct structured investigations, document findings, visualize relationships between entities, and collaborate in real time.

## Key features

| Feature | Description |
|---|---|
| **Investigations** | Organize work into independent investigations with statuses and collaborators |
| **Documents** | Rich editor with real-time collaboration (multiple simultaneous cursors) |
| **Tasks** | Assign and track tasks within each investigation |
| **Entities & Graph** | Create entities (people, IPs, domains…) and visualize their relationships |
| **Map** | Geolocate entities and events on an interactive map |
| **OSINT** | Quickly pivot on indicators of compromise (IoC) |
| **Templates** | Reuse document structures across investigations |
| **Calendar** | Visualize your task deadlines |

## Access and authentication

Log in with your username and password. On the first login created by an administrator, you will need to change your password.

Tracr has two roles:
- **User**: access to their own investigations and investigations where they are a collaborator
- **Administrator**: user management, access to all investigations, status and category management
