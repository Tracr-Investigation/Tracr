# Administration

The **Administration** panel is only accessible to users with the **Administrator** role.

## User management

### Create a user
1. Administration → Users → **New user**.
2. Enter the username, email, and role.
3. A temporary password is generated - the user will need to change it on their first login.

### Edit a user
Click on a user to edit their username, role, or force a password change.

### Disable an account
A disabled account can no longer log in. Associated data (investigations, documents) is preserved.

## Investigation status management

Administration → Investigation statuses. Create, rename, or delete the statuses available for investigations. Each status can have a color and an icon.

## Log categories

Administration → Log categories. Manage event categories used in investigation timelines.

## Template categories

Administration → Template categories. Organize document templates by category.

## Admin dashboard

The Administration panel home page displays global statistics: number of users, active investigations, and open tasks.
