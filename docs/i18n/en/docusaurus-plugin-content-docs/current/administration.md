# Administration

The **Administration** panel is only accessible to users with the **Administrator** or **Super-administrator** role.

---

## Admin dashboard

The Administration panel home page displays global statistics: number of users, active investigations, and open tasks.

---

## User management

### Create a user
1. Administration → Users → **New user**.
2. Enter the username, email, and role.
3. A temporary password is generated: the user will need to change it on their first login.

### Edit a user
Click on a user to edit their username, role, or force a password change.

### Reset a user's password
From a user's profile, the **Reset password** button generates a new temporary password and forces the user to change it on their next login.

### Disable an account
A disabled account can no longer log in. Associated data (investigations, documents) is preserved.

### Delete a user
The **Delete** button (trash icon) permanently deletes the account. This action is irreversible.

---

## Investigation status management

Administration → **Investigation statuses**. Create, rename, or delete the statuses available for investigations.

Each status has:
- A **name**
- A **color** (color picker)

---

## Investigation categories

Administration → **Categories**. Organize investigations by thematic categories.

Each category has:
- A **name**
- A **color**
- An **icon** (chosen from a list of Lucide icons)

---

## Log categories

Administration → **Log categories**. Manage event categories used in investigation timelines.

---

## Template categories

Administration → **Template categories**. Organize document templates by category.

---

## Activity log

Administration → **Logs**. View the history of all actions performed on the platform.

**Log features:**
- Text search in log details
- Filter by log category
- Option to hide read-only (consultation) actions
- Pagination (15 entries per page)
