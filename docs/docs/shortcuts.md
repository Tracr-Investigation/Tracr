# Raccourcis clavier

## Interface

| Raccourci | Action |
|---|---|
| `Ctrl` + `B` / `Cmd` + `B` | Replier / déplier la sidebar |

## Éditeur de documents

### Mise en forme du texte

| Raccourci | Action |
|---|---|
| `Ctrl` + `B` | Gras |
| `Ctrl` + `I` | Italique |
| `Ctrl` + `U` | Souligné |
| `Ctrl` + `Shift` + `X` | Barré |

### Structure

| Raccourci | Action |
|---|---|
| `#` + `Espace` | Titre H1 |
| `##` + `Espace` | Titre H2 |
| `###` + `Espace` | Titre H3 |
| `-` + `Espace` | Liste à puces |
| `1.` + `Espace` | Liste numérotée |
| `[ ]` + `Espace` | Case à cocher |

### Liens et navigation

| Raccourci | Action |
|---|---|
| `Ctrl` + `K` | Insérer / modifier un lien |
| `Ctrl` + `Z` | Annuler |
| `Ctrl` + `Shift` + `Z` | Rétablir |
| `Ctrl` + `A` | Sélectionner tout |

### Mentions

| Raccourci | Action |
|---|---|
| `@` | Mentionner une entité de l'enquête |

:::note
`Ctrl` sur Windows/Linux correspond à `Cmd` sur macOS.

Le raccourci `Ctrl+B` pour la sidebar n'est pas déclenché lorsque le curseur est dans l'éditeur, pour ne pas interférer avec le raccourci **Gras**.
:::
