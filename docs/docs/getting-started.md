# Prise en main

Ce guide vous accompagne de votre première connexion jusqu'à votre première enquête collaborative.

## 1. Première connexion

:::tip Codes de récupération
Après avoir changé votre mot de passe, configurez vos **codes de récupération** dans Paramètres → Sécurité. Ils vous permettront de récupérer votre compte si vous oubliez votre mot de passe.
:::

## 2. Créer votre première enquête

1. Depuis le **Tableau de bord**, cliquez sur **Nouvelle enquête**.
2. Donnez-lui un nom clair (ex. : "Incident réseau 2026-06").
3. Sélectionnez un statut (ex. : "En cours").
4. Validez.

Vous êtes maintenant le propriétaire de cette enquête.

## 3. Inviter un collaborateur

1. Ouvrez l'enquête que vous venez de créer.
2. Cliquez sur l'icône **Collaborateurs** en haut à droite.
3. Tapez le pseudo de votre collègue et ajoutez-le.

Il aura accès à tous les onglets de l'enquête dès maintenant.

## 4. Créer un premier document

1. Dans l'onglet **Documents**, cliquez sur **Nouveau document**.
2. Donnez-lui un titre.
3. Commencez à rédiger dans l'éditeur.

Si votre collègue ouvre le même document, vous verrez son curseur apparaître en couleur — la collaboration temps réel est active.

## 5. Ajouter des entités

1. Allez dans l'onglet **Graphe**.
2. Cliquez sur **Ajouter une entité**.
3. Renseignez le type (`ip`, `domaine`, `personne`…) et le libellé.
4. Pour relier deux entités, survolez le bord d'un nœud et tirez vers un autre.

## 6. Assigner une tâche

1. Dans l'onglet **Tâches**, cliquez sur **Nouvelle tâche**.
2. Renseignez le titre, assignez-la à un collaborateur et fixez une échéance.
3. Validez.

La tâche apparaît dans le **Calendrier** et le **Tableau de bord** de la personne assignée.

## Prochaines étapes

- Consultez la page [Documents](./documents.md) pour découvrir toutes les fonctionnalités de l'éditeur.
- Configurez vos préférences dans [Paramètres](./settings.md).
- Explorez l'[OSINT](./osint.md) pour enrichir vos indicateurs.
